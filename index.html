Customer Lifetime Value (CLV)

Calculates 5-year LTV for each customer, factoring in order history, average order value, and repeat rate. Used to identify high-value customers for retention campaigns.

SELECT customer_id, COUNT(DISTINCT order_id) as orders, ROUND(SUM(amount), 2) as ltv, ROUND(AVG(amount), 2) as avg_order FROM orders WHERE order_date >= DATE_SUB(CURDATE(), INTERVAL 5 YEAR) GROUP BY customer_id ORDER BY ltv DESC;
Impact: Identified top 20% of customers generating 60% of revenue, enabling prioritized retention efforts and personalized marketing campaigns.

RFM Segmentation

Segments customers into behavioral tiers (VIP, Loyal, At-Risk, Churned) based on recency, frequency, and monetary value for targeted marketing.

WITH rfm AS ( SELECT customer_id, PERCENT_RANK() OVER (ORDER BY recent_date) as r, PERCENT_RANK() OVER (ORDER BY freq_count) as f, PERCENT_RANK() OVER (ORDER BY total_spend) as m FROM customer_metrics ) SELECT customer_id, CASE WHEN r >= 0.8 THEN 'VIP' WHEN r >= 0.5 THEN 'Loyal' ELSE 'At-Risk' END as segment FROM rfm;
Impact: Enabled data-driven customer segmentation, reducing churn by 25% through targeted retention campaigns for at-risk segments.

Product Affinity Analysis

Identifies product pairs frequently purchased together using association rule mining, enabling cross-sell and recommendation strategies.

SELECT a.product_id as product_a, b.product_id as product_b, COUNT(*) as co_purchase_count, ROUND(COUNT(*) * 100.0 / (SELECT COUNT(DISTINCT order_id) FROM orders), 2) as percentage FROM orders a JOIN orders b ON a.order_id = b.order_id AND a.product_id < b.product_id GROUP BY a.product_id, b.product_id ORDER BY co_purchase_count DESC LIMIT 20;
Impact: Discovered high-affinity product pairs, increasing average order value by 18% through strategic product recommendations.

Cohort Retention Analysis

Tracks customer retention rates by acquisition month, measuring week-over-week and month-over-month cohort survival rates.

WITH cohorts AS ( SELECT customer_id, DATE_TRUNC('month', MIN(order_date)) as cohort_month FROM orders GROUP BY customer_id ) SELECT cohort_month, DATE_TRUNC('month', o.order_date) as month, COUNT(DISTINCT o.customer_id) as active FROM cohorts c JOIN orders o ON c.customer_id = o.customer_id GROUP BY cohort_month, month ORDER BY cohort_month, month;
Impact: Identified cohort with 85% 1-month retention vs. 60% average, revealing success factors for customer acquisition strategy.
