TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════

PROBLEM: Still seeing 404 error
SOLUTION:
1. Check all 11 files uploaded to ROOT (not in folder)
2. Files must be directly in repo (not in /files/ folder)
3. Wait 5 minutes and refresh

PROBLEM: GA4 not showing data
SOLUTION:
1. Check GA4 ID is correct (G-XXXXXXXXXX)
2. Check it's added to all 5 HTML files (2 times each)
3. Check files are uploaded with GA4 ID
4. Wait 30 seconds and refresh

PROBLEM: GSC verification fails
SOLUTION:
1. Check verification code is in index.html
2. Code must be BEFORE </head> tag
3. Wait 2 minutes and try again
4. Try refreshing the page in GSC

PROBLEM: Sitemap won't submit
SOLUTION:
1. Wait 1 minute after uploading robots.txt
2. Make sure sitemap.xml is uploaded
3. Try submitting again
4. Check robots.txt points to correct URL
