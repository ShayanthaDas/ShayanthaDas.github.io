repo: maketing

Marketing Campaign Attribution & Funnel Optimization

This analytics framework monitors end-to-end customer acquisition metrics, calculating conversion velocities across multi-tier promotional pipelines. It focuses on resolving structural inefficiencies in ad spend attribution modeling across paid acquisition channels.

README.md Snippet

### Core Project Objectives * Establish automated ETL processing chains using Python (Pandas/NumPy) to aggregate asynchronous campaign tracking datasets. * Standardize conversion path definitions across customer lifecycle stages to prevent ad spend attribution leaks. * Deliver unified BI interfaces via Advanced Excel, Power BI, and Tableau to slice campaign metrics by demographic cohorts and channel groups. ### Key Implementation Steps 1. **Data Cleaning & Engineering:** Cleaned marketing logs by eliminating null session values and transforming nested JSON strings into linear feature spaces. 2. **Attribution Modeling:** Formulated custom linear and first-touch attribution parameters via Python to distribute pipeline credit fairly.

repo: walmart

Walmart Retail Demand & Supply Chain Forecast Modeling

An enterprise retail data mining project assessing inventory movement, seasonal order spikes, and regional demand distributions across multiple storefront dimensions. This study identifies logistical gaps caused by holiday supply strain.

README.md Snippet

### Core Project Objectives * Design relational star-schema data models optimized for transactional retail logging systems across hundreds of distinct departments. * Calculate dynamic moving windows and rolling metric flags within SQL to capture macro demand changes during seasonal events. * Build complex, low-latency Tableau and Power BI visual layers with specific filters for asset deployment managers. ### Analytical Scope * **Demand Stability Centers:** Created logic to automatically highlight stockout risks when rolling demand exceeds standard stock buffer metrics. * **Financial Asset Tracking:** Evaluated gross profit margin patterns against promotion variables to map seasonal revenue drift.

repo: olist

Olist Brazilian E-Commerce Logistics & Fulfillment Infrastructure

An end-to-end data evaluation of marketplace performance, looking closely at shipping delays, customer retention cohorts, and geographical fulfillment bottlenecks using real commercial transactions.

README.md Snippet

### Core Project Objectives * Run relational analytical queries across massive, multi-table e-commerce schemas tracking customer vectors, seller hubs, item specifications, and payment terms. * Process temporal distribution statistics to map variances between estimated shipping delivery milestones and true final fulfillment arrivals. * Build interactive geographic heatmaps in Tableau along with retention matrices in Power BI to monitor market performance. ### Critical Insights Developed * Identified geographical delivery patterns showing specific cross-state shipping routes that accounted for disproportionate delivery delays. * Created customer cohort tracking models using advanced DAX and SQL to measure purchase intervals over multiple timeframes.
