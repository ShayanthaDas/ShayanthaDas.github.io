PART 3: Google Search Console Setup (5 minutes)
═════════════════════════════════════════════════════════════════════════════════

STEP 3.1: CREATE GSC PROPERTY
─────────────────────────────
1. Go to: search.google.com/search-console
2. Click "⬚ Create property" (or + Create property)
3. Select: "URL prefix"
4. Enter: https://shayanthadas.github.io
5. Click "Continue"
✅ Property created

STEP 3.2: VERIFY YOUR SITE
──────────────────────────
1. You'll see verification options
2. Select: "HTML tag"
3. Copy the code (something like: <meta name="google-site-verification" ...>)
4. Go to your GitHub repo → index.html
5. Click pencil icon (edit)
6. Find: </head> tag (near bottom)
7. Paste the verification code RIGHT BEFORE </head>
8. Example:
   
   <meta name="google-site-verification" content="xyz123abc...">
   </head>
   
9. Click "Commit changes"
10. Go back to GSC and click "Verify"
✅ Site verified!

STEP 3.3: SUBMIT SITEMAP TO GSC
───────────────────────────────
1. In GSC, click: "Sitemaps" (left sidebar)
2. In the text box, enter: sitemap.xml
3. Click "Submit"
4. You should see: "Sitemap successfully submitted"
✅ Sitemap submitted to Google!

STEP 3.4: MONITOR INDEXING
───────────────────────────
1. Go back to "Overview" in GSC
2. Wait 1-2 days for initial crawl
3. You'll start seeing:
   ✅ Pages indexed
   ✅ Impressions & clicks
   ✅ Search positions
✅ Google is indexing your site!

TROUBLESHOOTING
═════════════════════════════════════════════════════════════════════════════════

PROBLEM: Still seeing 404 error
SOLUTION:
1. Check files are in ROOT (not in folder)
2. Wait 5 minutes and refresh
3. Clear browser cache (Ctrl+Shift+Delete)
4. Try incognito/private mode

PROBLEM: Photo not showing
SOLUTION:
1. Check profile.jpg is uploaded
2. Refresh page
3. Check image filename is lowercase

PROBLEM: Styles look broken (no colors)
SOLUTION:
1. Wait 30 seconds and refresh
2. Clear cache
3. Hard refresh (Ctrl+Shift+R on Windows, Cmd+Shift+R on Mac)

PROBLEM: Contact form not working
SOLUTION:
1. Check internet connection
2. Verify email in form is correct
3. Check spam/junk folder for test messages
